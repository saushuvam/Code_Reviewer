# Code Reviewer 🚀

AI-powered code review platform built using React, Node.js, Express, and AI integration to help developers analyze, review, and improve code quality efficiently.

## Features

* AI-based code review and suggestions
* Clean and responsive UI
* Real-time code analysis
* Syntax highlighting
* Frontend built with React + Vite
* Backend built with Node.js + Express

## Tech Stack

### Frontend

* React
* Vite
* CSS
* Axios

### Backend

* Node.js
* Express.js

## Project Structure

```bash
Code_Reviewer/
├── frontend/
├── backend/
```

## Installation

### Clone Repository

```bash
git clone https://github.com/saushuvam/Code_Reviewer.git
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
npm install
node server.js
```

## Environment Variables

Create a `.env` file inside backend folder and add:

```env
GEMINI_API_KEY=your_api_key
```
